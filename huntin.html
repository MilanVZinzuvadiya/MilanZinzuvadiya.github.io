Huntin – Image Search with an Image

The AI Image Search and classification web Application

  • Programming Languages

    Python

  • Frameworks/libraries

    Tensorflow, OpenCV

  • Web Technologies

    HTML, CSS, JavaScript

Info

Huntin – A Image Search with an Image by AI, is a web application. It is developed to address the issue of object-based classification. From provided directory, it finds all images that contains object in the provided image. It is a demonstration of deep learning research into Computer Vision, which combines saliency and Siamese network to detect similar object in multiple images.
